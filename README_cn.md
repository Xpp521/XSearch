# XSearch
[[English version]](https://github.com/Xpp521/XSearch/blob/master/README.md "English version")
### 简介
XSearch可以让你更加快速、方便地完成网络搜索或者访问指定网站。还有一些附加的小功能。  
ጿ ኈ ቼ ዽ    ጿ ኈ ቼ ዽ    ጿ ኈ ቼ ዽ 
___
### 依赖
- PyQt5
- pynput
- requests
- PyHotKey
___
### 使用方法
双击大写锁定，然后开始你的表演。
