# XSearch
### Description
This tool can improve your search efficiency and prevent your device from sleeping.  
ጿ ኈ ቼ ዽ    ጿ ኈ ቼ ዽ    ጿ ኈ ቼ ዽ 

### Requirement
- PyQt5
- pynput
- requests
- PyHotKey

### Usage
Double click "Caps lock", start your search journey.
